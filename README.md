# WVTR — World Volleyball Team Rating

Cross-league rating system for professional volleyball clubs.

## Status

Early development.

## Stack

- Python 3.11+ (scraper, rating engine)
- Supabase / PostgreSQL (database)
- Vercel (frontend hosting)
- Next.js (frontend, tentative)
- GitHub Actions (scheduled scraper runs)

## Documentation

Methodology, BRD and technical specs live in `docs/`.
